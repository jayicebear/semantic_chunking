
# PDF Semantic Chunking  
**PDF 한 방에 → 의미 단위로 똑똑하게 쪼개진 고품질 청크**  
기업 내부 문서, 매뉴얼, 규정집, 데이터 거버넌스 가이드 PDF를 Docling MD 파일로 변환  
→ **Semantic Chunking**으로 RAG에 바로 넣을 수 있게 변환!

> 기존 fixed-size 청킹은 잊으세요. 이제는 **문맥이 끊기지 않는 진짜 의미 단위 청크**를 만듭니다.

## 핵심 알고리즘: 시맨틱 청킹 (Semantic Chunking) 원리

기존의 "N 자마다 자르기"는 문맥이 끊겨서 RAG 성능이 떨어집니다.  
이 프로젝트는 **문장 간 의미 유사도**를 실시간으로 계산해서, **진짜 주제가 바뀌는 지점에서만 청크를 나눕니다.**

## Embedding Model

`Qwen/Qwen3-Embedding-0.6B`

### 작동 방식 (Step-by-Step)

```text

ex) threshold 0.7

문장1 → 문장2 → 문장3 → 문장4 → 문장5 → 문장6
  │      │       │       │       │       │
임베딩 → 임베딩 → 임베딩 → 임베딩 → 임베딩 → 임베딩
  │      │       │       │       │       │
  └──────┴───▶ 유사도 0.82 → 같은 청크!
                  │
                  └──▶ 유사도 0.71 → 여전히 같은 청크!
                          │
                          └──▶ 유사도 0.38 → 주제가 바뀜! → 새 청크 시작
